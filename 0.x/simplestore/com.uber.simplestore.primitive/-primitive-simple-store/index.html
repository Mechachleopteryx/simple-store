    PrimitiveSimpleStore¶

    interface PrimitiveSimpleStore :SimpleStore

    Store primitives on disk.

    All methods never return null in the ListenableFuture, #`[SimpleStore#contains(String)`](../../com.uber.simplestore/-simple-store/contains.md) should be used for optionality. If the value is not set, the 0-byte primitive will be returned.
